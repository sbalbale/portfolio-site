02. Light Mode Usage Principles

Legibility Logic
visibility

Maximum Legibility

Light mode prioritizes readability in high-ambient environments. Every text element is calibrated for absolute clarity. Contrast between Background (#FFFFFF) and Primary Text (#1A1A1A) provides an optimal reading experience for long-form documentation.

shadows

Shadow Elevations

Depth is communicated through subtle tonal shadows rather than extreme background shifts. Use 'tonal-1' for cards and 'tonal-2' for interactive floating elements to maintain a clean, airy aesthetic.

Elevated Surface
grid_view

Structured Whitespace

In light mode, whitespace is a functional component. Use consistent gutters and surface-variant borders to define boundaries without heavy color blocking. The hierarchy is established by text weight and accent placement.
