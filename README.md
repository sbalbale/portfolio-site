## :dart: About

This repository houses the source code for my digital portfolio and centralized engineering monorepo. The platform serves as a high-performance delivery system for my ECE research, software projects, and professional experience.

Built as a strict Turborepo monorepo using pnpm, the architecture decouples the presentation layer from the data schema using a headless Sanity.io CMS. The deployment lifecycle is fully automated via GitHub Actions and optimized for sub-second execution through Vercel Remote Caching, Turbopack, and Next.js 16 React Server Components.

## :rocket: Technologies

The following tools were used in this project:

- [VS Code](https://code.visualstudio.com/)
- [PNPM](https://pnpm.io/)
- [Next.js 16](https://nextjs.org/) (App Router)
- [Sanity.io v3](https://www.sanity.io/) (Headless CMS)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/) (Animations)
- [TypeScript](https://www.typescriptlang.org/)
- [Lucide React](https://lucide.dev/) (System Icons)
- [Vercel](https://vercel.com/)
- [GitHub](https://github.com/)
- [Sanity.io](https://www.sanity.io/)
- [Github Actions](https://github.com/features/actions)

## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/sbalbale/portfolio-site

# Access
$ cd portfolio-site

# Install dependencies
$ pnpm install

# Run
$ pnpm dev
```
