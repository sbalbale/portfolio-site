## :dart: About

This is a high-performance, headless CMS-driven portfolio architecture.

The project is managed as a **PNPM Monorepo**, separating the high-fidelity Next.js frontend from the Sanity.io Studio backend.

## :rocket: Technologies

The following tools were used in this project:

- [Next.js 14](https://nextjs.org/) (App Router)
- [Sanity.io v3](https://www.sanity.io/) (Headless CMS)
- [Tailwind CSS](https://tailwindcss.com/) (Material High Contrast)
- [Framer Motion](https://www.framer.com/motion/) (Architectural Animations)
- [TypeScript](https://www.typescriptlang.org/)
- [Lucide React](https://lucide.dev/) (System Icons)

## :checkered_flag: Starting

```bash
# Clone this project
$ git clone [https://github.com/sbalbale/sean-balbale-portfolio](https://github.com/sbalbale/sean-balbale-portfolio)

# Access the root directory
$ cd sean-balbale-portfolio

# Install dependencies for the entire monorepo
$ pnpm install

# Set up environment variables in /apps/web and /apps/studio
# (Refer to Phase 31 documentation for .env requirements)

# Run development environment
$ pnpm dev
```
